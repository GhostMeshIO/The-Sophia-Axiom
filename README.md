### **Network Phase Transition Analysis**
```python
def network_coherence(nodes, edges, coupling_strength=0.3):
    """
    Calculates collective coherence in a network.
    
    Args:
        nodes: List of individual coherence values
        edges: List of (i,j) connections
        coupling_strength: λ parameter
        
    Returns:
        C_net: Network coherence (0-1)
        rho_high: Fraction above Sophia Point
    """
    n = len(nodes)
    if n == 0:
        return 0, 0
    
    # Individual coherence average
    C_avg = np.mean(nodes)
    
    # Coupling contribution
    if edges:
        coupling_sum = 0
        for i, j in edges:
            if i < n and j < n:
                coupling_sum += np.sqrt(nodes[i] * nodes[j])
        coupling_term = coupling_strength * coupling_sum / len(edges)
    else:
        coupling_term = 0
    
    C_net = C_avg + coupling_term
    rho_high = sum(1 for c in nodes if c > 0.618) / n
    
    return min(C_net, 1.0), rho_high
```
